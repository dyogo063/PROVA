1 - O que é clone?
CLONE SERVE PARA FAZER UMA COPIA DO REPOSITÓRIO REMOTO PARA O REPOSITÓRIO LOCAL NA PRIEMRA VEZ EM QUE FOR 
ACESSADO O REPOSITÓRIO LOCAL

2 - O que é commit?
É O EMPACOTAMENTO DO ARQUIVO

3 - O que é um repositório?

REPOSITÓRIO É A PASTA ONDE GUARDA TODOS OS ARQUIVOS DO PROJETO

5 - O que é pull?
O PULL PUXA TODAS AS INFORMAÇÕES DO REPOSITÓRIO REMOTO PARA O REPOSITÓRIO LOCAL
